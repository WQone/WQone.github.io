## 第一次开始尝试学习 react，集成 React + Dva + Antd + umi 快速搭建一个项目

<!--more-->

#### 全局环境下安装 Umi

```bash
$ npm install -g umi

umi 提供了脚手架供我们快速创建项目
```

#### 进入文件夹,使用 create umi 搭建项目基础框架

```bash
$ cd react-test-demo
$ npm create umi
```

#### 选择需要的功能进行安装：antd,dva

### 回车确认后，文件和目录就创建好了

```bash
$ npm install
$ npm start

打开http://localhost:8080就可以看到了
```
